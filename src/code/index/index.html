International Code of Phylogenetic Nomenclature (PhyloCode)

Index

The references are not to pages but to the Articles, Recommendations, etc. of this Code, as follows: Pre = Preamble; Pri = Principles; Numerals = Articles; Numerals followed by letters = Recommendations; Ex = Examples; N = Notes; App = Appendix; G = Glossary. The Preface is not covered by the Index.

If a term occurs in both an article, recommendation, or note and an accompanying example, often only the article, recommendation or note is cited. However, if the term appears in an example but not the article, recommendation, or note with which it is associated, the example is cited (e.g., 9.11.Ex1).
